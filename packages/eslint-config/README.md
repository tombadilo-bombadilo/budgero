# `@budgero/eslint-config`

Shared ESLint (flat config) for the Budgero monorepo: Airbnb (base + TypeScript) with
React additions, Prettier compatibility, file/folder naming rules, and the
Feature-Sliced Design layer-boundary rules used by `packages/app`.

> Part of the Budgero monorepo. **Source-available** under [FSL-1.1-ALv2](../../LICENSE).

## Entry points

| Import | Use |
|---|---|
| `@budgero/eslint-config/base` | TypeScript packages (core, runtime) |
| `@budgero/eslint-config/react` | React packages (app) |
