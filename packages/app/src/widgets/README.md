# `widgets/` — FSD layer

Large **composite UI blocks** that assemble entities + features into a
self-contained unit: dashboard cards, sidebar / mobile nav, command palette,
transaction table, budget table, chat panel.

- **Alias:** `@widgets/*`
- **May import from:** `features`, `entities`, `shared`.
- **Imported by:** `pages`, `app`.

See [Feature-Sliced Design](https://feature-sliced.design/).
