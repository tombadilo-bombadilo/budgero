# `pages/` — FSD layer

Thin **route-level compositions**. A page wires widgets/features/entities together
for a route and holds little logic of its own.

- **Alias:** `@pages/*`
- **May import from:** `widgets`, `features`, `entities`, `shared`.
- **Imported by:** `app`.

See [Feature-Sliced Design](https://feature-sliced.design/).
